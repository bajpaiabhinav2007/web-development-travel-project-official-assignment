## Wanderlust Travel Agency Website:

Wanderlust is a modern, responsive travel agency landing page built using **HTML5** and **CSS3**.  
It showcases popular destinations, tour packages, testimonials, blogs, and contact information in a visually appealing way.

-------------------------------------------------------------------------------------------------------------------------

##  Project Overview:

This project is a static travel website designed to:
- Promote travel destinations
- Display tour packages
- Share customer testimonials
- Publish travel blogs
- Allow users to contact the agency and subscribe to a newsletter
